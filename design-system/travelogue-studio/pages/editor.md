# Editor Override

- Optimize for repeated work, not marketing presentation.
- Use one compact command header and edge-aligned page toolbar.
- Desktop tool rail may expose specialist tools; mobile groups them under Style.
- Keep project title, undo, redo, preview, and export reachable without opening a panel.
- Inspector sections use dividers inside one panel instead of nested cards.
- Opening a photo or frame selects the contextual Image tools panel automatically.
- Admin-only capabilities stay role-gated and visually distinct.
